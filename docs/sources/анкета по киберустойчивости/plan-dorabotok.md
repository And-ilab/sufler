# План мероприятий по устранению замечаний (анкета киберустойчивости)

**Версия:** 1.1 · **Дата:** 2026-06-24 · **Проект:** ПО на базе искусственного интеллекта для банковских процессов · **Договор:** № 14-03/2026

---

## 1. Контекст и назначение

### 1.1. Контекст

ООО «ГС Ритейл» по результатам оценки киберустойчивости, проведённой ОАО «АСБ Беларусбанк», получило **26,5 балла** — оценка **«неудовлетворительно»**. Заказчик указал на отсутствие единой политики ИБ, аттестата СЗИ и формализованных процессов (контроль утечек, уязвимости, почта, инциденты).

Настоящий план описывает **мероприятия**, **сроки**, **ответственных** и **связанные документы** для устранения замечаний и подготовки к повторной оценке.

### 1.2. Назначение

Координировать работу команды проекта «ПО на базе искусственного интеллекта для банковских процессов» по разработке, утверждению и внедрению документов ИБ.

---

## 2. Цели

1. Закрыть **шесть ключевых замечаний** банка документально и организационно.
2. Подготовить **комплект подтверждающих материалов** для повторной подачи анкеты.
3. Обеспечить **единую структуру** политик и регламентов (глоссарий, роли, цели, задачи, процедуры).
