# Документация Active Directory (заказчик)

Исходные материалы от банка по интеграции с Active Directory: схема групп, политики, инструкции LDAP/SSO.

> Папка ожидает файлы от заказчика. Требования в договоре — в [prilozhenie-1.md](../technical-requirements/prilozhenie-1.md) (§5.1.34, §7.1).

## Ожидаемые типы документов

- Схема групп AD и матрица ролей
- Политики аутентификации (LDAP, Kerberos, SSO)
- Инструкции по выдаче учётных записей и групп для ПО
- Контакты ответственных со стороны ИБ / инфраструктуры
