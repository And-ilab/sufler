## Правила

- Сюда кладутся только **входящие** материалы заказчика; не дублировать наши `tz-*.md`.
- Технические `протокол-интеграция-*.md` в `integration/` — **не** протоколы совещаний.
- Имена файлов: латиница, kebab-case; для протоколов встреч — `YYYY-MM-DD-тема.md`.
