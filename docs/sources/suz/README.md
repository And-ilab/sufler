# Документация СУЗ (заказчик)

Исходные материалы от банка по **Системе управления знаниями** (СУЗ на базе 1С-Битрикс: Система управления сайтом): структура разделов, форматы статей, регламенты публикации, доступы.

> Это **не** наше ТЗ на интеграцию. Рабочий документ Исполнителя — [tz-bitrix-rag-sufler.md](../../integration/suz-bitrix-rag/tz-bitrix-rag-sufler.md).

## Ожидаемые типы документов

- Руководство пользователя / администратора СУЗ
- Схема разделов и рубрик базы знаний
- Описание полей статей, шаблонов и workflow публикации
- Регламент актуализации контента и ответственные редакторы
- Доступы к тестовому / продуктивному стенду Bitrix CMS
- Webhook-документация или API СУЗ (если предоставлено заказчиком)
