## Вкладка «Ассистент»

- Select базы знаний, «+ Новый», история диалогов
- Лента: вопрос / ответ, блок «Источники» (% + «Открыть»)
- Обратная связь: «Полезно» / «Неполный» / «Неверно» (tooltip по Прил.1); при «Неполный» — уточняющий запрос
- Прикрепить, инструменты, поле ввода, **Отправить**

## Вкладка «Документы»

Sub-tabs: **Очередь** · **Загрузить** · **Проверка**

| Sub-tab | Элементы |
|---------|----------|
| Очередь | Фильтры, таблица (файл, тип, статус), progress при processing, **Открыть** |
| Загрузить | Drop-zone, тип документа, пакет, **Начать распознавание** |
| Проверка | Split: viewer + bbox, поля + confidence, LLM-правка, экспорт JSON/XML/CSV, **Утвердить и экспорт** |

## Вкладка «Суфлёр» (канал телефония · Oktell)

**Принцип (FR-SUF-13):** лаконичный UI — без перегрузки; оператор читает подсказку за секунды.

| Зона | Содержание |
|------|------------|
| Шапка | Бейдж «Звонок», режим Консультация / Услуга, «Свернуть» (§4.6.2.3.1) |
| Лента пар | Вертикальный список блоков **«Клиент → Подсказка»** — одна рамка на пару, последняя реплика выделена |
| Подсказка | Краткий текст (≤ 500 симв.), % релевантности, «СУЗ ↗» одной строкой |
| Обратная связь | Три кнопки под подсказкой: **Воспользовался / Не воспользовался / Неполный** (§4.3.1.9) |

**Убрано с телефонии:** отдельный блок «контекст», полный транскрипт, кнопки «Отправить / Редактировать / Отклонить» (оператор отвечает **голосом**, §4.3.1).

- **Не отправляет ответ клиенту** — оператор отвечает голосом по телефону (§4.3.1)
- Подсказка LLM: текст на экране + **TTS оператору в наушник** (§4.3.1.11)
- **На каждую реплику клиента** — не более **2** подсказок (FR-SUF-11); привязка к строке §4.6.2.2.2
- **Режим активного звонка:** tab bar Hub фиксирован на «Суфлёр» (FR-SUF-12)

> **Онлайн-чат** — отдельный АРМ (`online-chat-mockups.canvas.tsx`): там допустима вставка текста в поле ответа, не TTS.

## Демо в Canvas

1. Откройте `ai-hub-panel-mockup.canvas.tsx` — превью справа от чата.
2. Кнопки **Демо: Ассистент / Документы / Суфлёр** — сразу нужная роль и вкладка.
3. Для **всех состояний вкладки «Ассистент»** (пустой чат, стриминг, ошибка, генерация doc) — `ai-assistant-ui-mockup.canvas.tsx`.
4. **Все вкладки (админ)** — переключение между тремя модулями без смены роли вручную.
