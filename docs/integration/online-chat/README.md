## Как согласовать

1. Прочитать **§1–2** ТЗ (контекст, глоссарий, роли).
2. Вставить **слайды 1–6** в **§4** из Online Chat Mockups; отметить **§4.6**.
3. Пройти **§5** — пользовательские сценарии (UC).
4. Согласовать **§6–7** (функционал и состав поставки).
5. Ответить на **§9**; зафиксировать протокол встречи.

## Экспорт в Word / Google Docs

### 1. Установить Pandoc (один раз)

```powershell
winget install --id JohnMacFarlane.Pandoc -e
```

Проверка: `pandoc --version`

### 2. Скрипт `export-to-word.ps1`

```powershell
cd c:\sufler\docs\integration\online-chat
.\export-to-word.ps1
```

Результат: `TZ-online-chat.docx`. Другой путь: `.\export-to-word.ps1 -OutputPath "D:\TZ.docx"`

### 3. Одной командой (без скрипта)

```powershell
cd c:\sufler\docs\integration\online-chat
pandoc tz-online-chat-platform.md -o TZ-online-chat.docx --standalone --toc --toc-depth=3
```

### 4. После экспорта

- **§4** — вставить скриншоты слайдов 1–6 из `canvases/online-chat-mockups.canvas.tsx` вместо «Место для слайда N».
- **Mermaid** (§3, §4.2, §8) — в Word не рендерится; при необходимости PNG с [mermaid.live](https://mermaid.live).
